# Simple To‑Do App (HTML, CSS, JavaScript)

A small, clean to‑do list app built with **vanilla JavaScript**, **HTML**, and **CSS**.  
No frameworks, no build tools – just classic front‑end code for learning and practice.

## Features

- Add new tasks
- Mark tasks as done / not done
- Delete individual tasks
- Clear all completed tasks
- Task counter (total and open)
- Saves tasks in `localStorage` so they survive page reloads
- Minimal, modern UI

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

.
├── index.html   # Markup / page structure
├── style.css    # Styling and layout
└── app.js       # To‑do app logic
